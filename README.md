## 4. 금융 트렌드 분석 프로세스

### 4.1 분석 시점

| 주기 | 시점 | 분석 깊이 |
|:----:|:----:|:----------|
| **월간** | 매월 1일 | 간략 (시장 동향만) |
| **분기** | 1/4/7/10월 | 상세 (섹터별 전망) |
| **연간** | 12월 | 심층 (전략 재검토) |

### 4.2 분석 항목 체크리스트

#### A. 거시경제 지표

| 분류 | 지표 | 확인 포인트 |
|------|------|-------------|
| **금리** | 미국 기준금리 | 인상/동결/인하 방향 |
| | 한국 기준금리 | 금통위 결정 |
| | 미국 10년물 금리 | 장단기 금리차 |
| **환율** | 원/달러 환율 | 1,300원 기준 고저 판단 |
| **인플레이션** | 미국 CPI/PCE | 2% 목표 대비 |
| | 한국 CPI | 물가 안정 여부 |

#### B. 주식시장 전망

| 시장 | 확인 지표 | 의미 |
|------|----------|------|
| **미국** | S&P500 전망 | 월가 컨센서스 |
| | 나스닥 전망 | 기술주 방향성 |
| | VIX 지수 | 변동성/공포 수준 |
| **한국** | 코스피 전망 | 증권사 목표치 |
| | 코스닥 전망 | 중소형주 방향 |
| | 외국인 수급 | 순매수/순매도 추이 |

#### C. 섹터/테마 동향

| 테마 | 확인 사항 |
|------|----------|
| **AI/반도체** | 엔비디아 실적, 데이터센터 투자 |
| **로봇** | 테슬라 옵티머스, 휴머노이드 뉴스 |
| **배당주** | 고배당 섹터 트렌드 |
| **채권** | 금리 방향에 따른 채권 가격 |

### 5.4 리밸런싱 실행 방법

#### 방법 A: 매도-매수 방식 (큰 조정)
```
1. 초과 비중 펀드 일부 매도
2. 부족 비중 펀드로 매수
3. 목표 비율 복원
```

#### 방법 B: 납입금 조정 방식 (작은 조정)
```
1. 월 납입금을 부족 비중 펀드에 집중
2. 초과 비중 펀드는 납입 중단
3. 자연스럽게 비율 복원 (2-3개월 소요)
```

### 5.5 펀드 교체 검토 기준

| 조건 | 기준 | 조치 |
|------|------|------|
| **수익률 부진** | 벤치마크 대비 -10%p 이상 (1년) | 대안 펀드 검토 |
| **테마 소멸** | 투자 테마 유효성 상실 | 신규 테마 펀드로 교체 |
| **운용사 이슈** | 펀드매니저 교체, 운용 방침 변경 | 모니터링 강화 |
| **비용 증가** | 총보수 인상 | 저비용 대안 검토 |
